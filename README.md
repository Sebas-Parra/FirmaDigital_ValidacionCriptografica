# FirmaDigital - Validación Criptográfica

Plataforma web segura de firma digital con validación criptográfica.

## Stack
- **Backend:** Python + Flask + SQLAlchemy + Supabase
- **Frontend:** Vue.js 3
- **Base de datos:** PostgreSQL (Supabase)
